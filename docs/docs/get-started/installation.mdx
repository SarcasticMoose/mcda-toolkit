# Installation

### What you'll need

- [.NET SDK](https://dotnet.microsoft.com/en-us/download/visual-studio-sdks) version 8.0 or above

:::info

.NET Standard 2.1 is supported implicitly. Please check compatibility with your target version in the [documentation](https://learn.microsoft.com/en-us/dotnet/standard/net-standard?tabs=net-standard-2-1)
:::
