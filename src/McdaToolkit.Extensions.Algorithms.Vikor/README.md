# McdaToolkit.Extensions.Vikor

VIKOR extension for the MCDA Toolkit providing an implementation of the VIKOR (VlseKriterijumska Optimizacija I Kompromisno Resenje) method.

This package extends the core toolkit with a complete VIKOR decision-making algorithm, including result computation and ranking conversion.

---

## Installation

```bash
dotnet add package McdaToolkit.Extensions.Vikor
